# Amitesh Mogha - Interactive Portfolio

Welcome to my personal portfolio! This web application is designed with a premium, Netflix-inspired dark aesthetic, featuring multiple unique user profiles (Recruiter, Developer, Stalker) to explore my journey, skills, projects, and personal interests.

## 🌟 Features

- **Profile Selection System**: Like Netflix, choose "Who is watching?" to view different tailored content layouts.
- **Glassmorphic UI**: High-end modern CSS glass effects, dark gradients, and smooth hover animations.
- **Interactive Timelines**: Scrollable and collapsible vertical & horizontal timelines of my professional and educational journey.
- **Dynamic Content Grids**: Sections for Skills, Certifications, Music, and Books that toggle seamlessly between horizontal scroll and fully expanded grid layouts.

## 🚀 Tech Stack

- **React** (TypeScript)
- **Vite** (Build Tool)
- **CSS3** (Custom Animations & Layouts)
- **React Router** (Navigation)
- **React Vertical Timeline Component**
- **Sanity** (Headless CMS - *configured in codebase*)

## 💻 Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/CodeWithMogha/NETFLIX-PORTFOLIO-.git
   cd NETFLIX-PORTFOLIO-
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
